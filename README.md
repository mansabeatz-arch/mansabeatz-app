# Mansabeatz App

Welcome to the Mansabeatz App! This is a modern web application built with HTML, CSS, and JavaScript.

## Features

- Responsive design
- Smooth navigation
- Clean and modern UI

## Getting Started

1. Clone this repository
2. Open `index.html` in your web browser
3. Start customizing the app!

## File Structure

- `index.html` - Main HTML file
- `style.css` - Stylesheet
- `script.js` - JavaScript functionality
- `README.md` - This file

## Development

Feel free to modify the files to suit your needs. The app uses vanilla HTML, CSS, and JavaScript for simplicity and performance.
